# 自动驾驶logsim和wordsim的区别

> 导入自 Claude AI · 2026-04-15 · 理论概念           

- LogSim（日志仿真） 和 WorldSim（世界仿真） 是自动驾驶仿真系统中两种不同的仿真模式：                                                                                                                         
## LogSim（日志回放仿真）

```
- 原理：回放真实采集的传感器数据（激光雷达点云、摄像头图像、GPS/IMU等）                                      
- 特点：                                                                                                     
  - 数据来自真实世界，确定性强                                                                               
  - 无法改变场景（ego 车辆行为固定）                                                                         
  - 适合测试感知、定位、预测模块                                                                             
  - 速度快，成本低                                                                                           
- 局限：无交互性，其他车辆行为不会响应 ego 车的决策                                                          
```

## WorldSim（世界仿真）

```
- 原理：构建完整的虚拟世界，包含物理引擎、交通流、传感器模型等                                               
- 特点：                                                                                                     
  - 场景可交互——其他 agent 会响应 ego 车的决策                                                               
  - 可生成无限变体场景（corner case 测试）                                                                   
  - 适合测试规划、控制、决策模块                                                                             
  - 支持闭环测试（closed-loop）                                                                              
- 局限：传感器模型不够真实（sim-to-real gap），计算成本高                                                    
```

## 核心区别对比

| 维度 | LogSim | WorldSim |
| --- | --- | --- |
| 数据来源 | 真实采集 | 虚拟生成 |
| 闭环能力 | 开环（open-loop） | 闭环（closed-loop） |
| 交互性 | 无 | 有 |
| 真实性 | 高 | 低（依赖模型精度） |
| 覆盖场景 | 受采集限制 | 理论无限 |
| 典型用途 | 感知/预测回归 | 规划/控制验证 |

## 1. LogSim 做感知模块的快速回归测试
